# Design System (Personal)

Minimalism is not aesthetic. It is discipline.

### Color System

```css
--color-bg: #FAFAFA;
--color-text-primary: #0A0A0A;
--color-text-secondary: #6B7280;
--color-accent: #2563EB;      /* subtle blue */
--color-accent-soft: #E0E7FF;
--color-border: #E5E7EB;
```

### Typography

* Primary: System UI (-apple-system, Inter)
* Secondary: Neutral Sans
* Mono: JetBrains Mono

### Spacing System

* Section spacing: 96px – 128px
* Content max-width: 1100px
* Grid: 2-column asymmetric

### Principles

* White space is structural, not decorative
* Typography carries hierarchy
* Color is used sparingly for meaning
